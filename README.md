#  Recipe Finder & Meal Planner Web App

# Project Overview

The Recipe Finder & Meal Planner is a web-based application that allows users to search for recipes using keywords or ingredients. The application fetches real-time data from a public API and displays relevant recipes with images, categories, and cooking instructions. It also provides filtering and sorting features to enhance user experience.



# Purpose of the Project

The main purpose of this project is to demonstrate the understanding of JavaScript concepts such as:

* API integration using the Fetch API
* Use of higher-order functions like map, filter, and sort
* Building an interactive and user-friendly interface using HTML and CSS



# API Used

**TheMealDB API**
https://www.themealdb.com/api.php

This API provides free access to a large collection of meal recipes, including details like ingredients, instructions, images, and categories.



# Technologies Used

* HTML for structuring the web page
* CSS for styling and layout
* JavaScript (ES6) for functionality
* Fetch API for retrieving data from the public API



# Features

* Search recipes by meal name or ingredient
* Display recipe cards with images and basic details
* Filter recipes based on category (e.g., vegetarian, dessert, etc.)
* Sort recipes alphabetically
* View detailed cooking instructions for each recipe



# How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open the `index.html` file in any web browser
4. Enter a recipe name or ingredient in the search bar
5. Browse through the results
